Hướng dẫn Restore PostgreSQL

- Cài đặt PostgreSQL
- Mở pgadmin4
- Click chuột phải vào Databases -> Create -> Database (Hộp thoại create - Database hiện lên)
 	+ Điền tên cho database CoffeeManager
	+ Save
-> Database CoffeeManager sẽ được tạo
- Click chuột phải vào CoffeeManager -> Restore (Hộp thoại restore hiện lên)
+ Filename: Chọn đến file backup có tên postgreSQL.backup
+ Role name: chọn postgres
- Chọn Restore
